HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The different header elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or a block of text.
  • The ul element represents unordered lists
  • The ol element represents ordered lists
  • The li element represents list items
  • The img element contains information about the images displayed on the webpage.
  • The src attribute defnes the location of the image
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element, also known as an anchor, creates links to the same webpage or other webpages
  • The href attribute points to the URL for the link
  • The alt attribute contains a string text that describes the appearance and functionality of the image.

CSS

  • CSS:Gives the content style
  • There are 3 ways to style a webpage using CSS: inline CSS, internal CSS, and external style sheet.
  • Inline CSS: to style an element directly in the HTML
  • -Apply to a specific HTML tag using a style attribute with a CSS rule to style a particular page element only.
  • -These are useful for quick and permanent changes, but less flexible.
  • REASON: each line must be edited individually when you make a design change.
  • -style attribute changes the color of a single heading
  • Internal Style Sheet is used to embed an entire style sheet use style element in the head element.
  • -CSS rules for the head section of the HTML file only apply to that page.
  • External Style Sheet is the most common way to style a webpage.
  • -it allows developers to keep all their rules in a separate file, making design changes easier.
  • -we link the file using a link element placed in the HTML's head element
  • *Seperation of concerns: each section of code should have it's own responsibility
  • margin property acts much like a margin in a word processing document
  • -indicates how much space we want around the outside of an element
  • -when we set the margin property to 0 we are indiciating we want no extra space around the element
  • padding property adds space inside the content of an element
  • rel: is for the relationship between the current document and link document
  • href: the location (URL) of the external resource

Git

  • Git is a version control system that lets you manage and keep track of each version of your project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The "add" command adds modifications in the current working branch to the staging area.
  • The "-A" indicates that we want to add all changes
  • git commit: a command with a commit message that describes what the commit contains.
  • git pull margin: Double checks if our local branch is insync with thte base branch in GitHub
